Для создания директорий, как видно на скриншоте, я использовал команду mkdir. 
На другом скриншоте я использовал параметр -р для создания директории одновременно с поддиректорией. 
Для перемещения файлов использовал команду mv, несколько раз с параметром -v для отображения процесса. 
Все остальные команды: touch, cp, cat использовал без параметров из-за ненадобности. 
Так же во время пробных попыток использовал команду rm стакими параметрами как -r для удаления директорий 
и с параметром -i (всегда использовал данный параметр, как и было посоветовано на лекции) для подтверждения удаления.
